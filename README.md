# 淘宝双11销售趋势与用户行为分析

> GMV 真实公开数据 + 用户行为模拟数据 | 电商数据分析与机器学习方法演示

## 数据说明

| 数据类型 | 来源 | 说明 |
|---|---|---|
| GMV 数据 | 2009-2023 年淘宝双 11 公开销售额 | 真实公开数据 |
| 用户行为数据 | 基于业务规则模拟生成 | 50,000 条样本，用于演示 RFM、聚类、分类建模方法 |

**注意**：由于用户行为数据为模拟数据，本项目中的机器学习模型结果（AUC、特征重要性等）仅用于展示算法建模流程，不代表真实业务预测能力。

## 核心内容

- **销售趋势分析**：2009-2023 年 GMV 增长趋势、增长率变化
- **用户画像分析**：年龄、性别、城市层级、消费分层
- **回头客预测**：逻辑回归 / 随机森林分类模型，支持特征选择与交互训练

## 技术栈

- Python
- Streamlit
- Pandas / NumPy
- scikit-learn

## 运行方式

```bash
pip install -r requirements.txt
streamlit run taobao_11_streamlit.py
```

## 项目定位

本项目为**数据分析与机器学习方法演示项目**，重点展示：
- 时序趋势可视化
- 用户分层与 RFM 思路
- 分类模型训练与评估流程
